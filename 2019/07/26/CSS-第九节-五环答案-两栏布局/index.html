五环题目答案

解题思路

  1. 五个环=5个div 并且由一个盒子包围
  2. 先让五个环可以定位,将其定位到合适位置
  3. 让外围div包围五环并带他到居中位置(根据盒子大小移动margin)
  4. 去掉外围div的辅助材料

css中注释部分为辅助材料

两栏布局

  1. 让两个div变成一栏,需要让其中一个变成position:absolute;

  2. 内容不允许覆盖,所以要用一个div的margin把另一个推开.
